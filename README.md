# ☕ CoffeeShop UI - Premium Experience

A production-ready, feature-rich Coffee Shop application built with **Flutter**. This project has been transformed from a simple UI slicing into a fully dynamic application using the **Provider** pattern, featuring smooth animations and a premium user experience.

---

## ✨ Features

### 🚀 Premium Experience
- **Hero Animations:** Seamless image transitions from catalog to detail pages.
- **Lottie Animations:** Interactive success animations for a delightful user journey.
- **Dynamic State Management:** Powered by `Provider` for real-time updates across the app.

### 🛒 Advanced E-Commerce Logic
- **Customization Engine:** Choose your size (S/M/L), milk type (Oat, Almond, etc.), and add extra shots.
- **Dynamic Pricing:** Real-time price calculation based on selections.
- **Shopping Cart:** Full CRUD operations for cart items.
- **Order Tracking:** Real-time-like tracking system after every purchase.

### 🔍 Smart Discovery
- **Real-time Search:** Filter the entire coffee catalog instantly.
- **Category Tabs:** Browse by coffee types with a silky smooth interface.

### 🎁 Loyalty & Profile
- **Reward System:** Earn points on every purchase and track your progress.
- **Order History:** Detailed log of all past transactions.
- **Favorites System:** Save your preferred brews for quick access.

---

## 📱 App Structure

The app consists of over 25+ production-grade screens, including:

- **Auth:** Login, Register, Forgot Password.
- **Main:** Home Catalog, Dynamic Search, Favorite List.
- **Order Flow:** Product Details, Customization, Cart, Checkout, Success Page, Tracking.
- **User:** Profile Management, Order History, Rewards Center, Settings.
- **Info:** About Us, Store Locator, Support/Help Center.

---

## 🛠️ Tech Stack

- **Framework:** Flutter (Stable Channel)
- **State Management:** Provider
- **Animations:** Lottie, Hero, TweenAnimation
- **Typography:** Google Fonts (Inter/Outfit)
- **Icons:** Cupertino & Material Icons

## 🚀 Getting Started

### Prerequisites
- Flutter SDK installed on your machine.
- An IDE (VS Code or Android Studio).

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/rizky28eka/CoffeShop-UI.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd CoffeShop-UI
   ```

3. **Install dependencies:**
   ```bash
   flutter pub get
   ```

4. **Run the application:**
   ```bash
   flutter run
   ```
